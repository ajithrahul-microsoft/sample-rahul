# sample-rahul

## Add Eight Numbers

This repository contains a Python script to add 8 numbers entered by the user.

### Usage

Run the script:
```bash
python3 add_eight_numbers.py
```

The script will prompt you to enter 8 numbers one by one, and then display their sum.

### Features
- Accepts integers and decimal numbers
- Input validation with error handling
- Clear prompts and output formatting
